# algorithim-Repo

1. Step 1. Start
1. Step 2. Input A, B, C
1. Step 3. Set Max to A
1. Step 4. If B > Max, set Max to B
1. Step 5. If C > Max, set Max to C
1. Step 6. Output Max
1. Step 7. End
